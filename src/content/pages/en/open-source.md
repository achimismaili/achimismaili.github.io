## Open Source Projects

### easy-web

Shared Astro component library for the ismaili.de web ecosystem. Reusable components, CSS design tokens, CI/CD pipelines for Azure Static Web Apps.

**Features:**
- 4 header variants (Classic, Centered, Hide-on-Scroll, Flyout)
- Gallery system with 6 variants
- Multilingual support (DE/EN)
- Light/dark mode with CSS tokens

**Live demos:**
- [Templates showcase](/templates/) — 4 homepage templates
- [Header variants](/nav-variants/) — 4 navigation styles
- [Components overview](/components/)

*Documentation and full demo project coming on GitHub Pages.*
