## Open-Source-Projekte

### easy-web

Gemeinsame Astro-Komponentenbibliothek für das ismaili.de Web-Ökosystem. Wiederverwendbare Komponenten, CSS Design-Tokens, CI/CD-Pipelines für Azure Static Web Apps.

**Features:**
- 4 Header-Varianten (Classic, Centered, Hide-on-Scroll, Flyout)
- Galerie-System mit 6 Varianten
- Mehrsprachigkeit (DE/EN)
- Light/Dark Mode mit CSS-Tokens

**Live-Demos:**
- [Vorlagen-Showcase](/templates/) — 4 Startseiten-Templates
- [Header-Varianten](/nav-variants/) — 4 Navigation-Stile
- [Komponenten-Übersicht](/components/)

*Dokumentation und vollständiges Demo-Projekt folgen auf GitHub Pages.*
